Mathematical Formula Sandbox
1. Multivariate Anomaly Score (Z-Distance)

Computes deviations from standard baseline parameters, acting as a lightweight Mahalanobis distance without covariance cross-terms:

D(x) = √ [ 1/N * Σ ( (x_i - μ_i) / σ_i )² ]
Where:
- x_i is the active sensor reading.
- μ_i is the baseline normal sensor mean.
- σ_i is the sensor standard deviation.
- N is the count of sensors.
2. Remaining Useful Life (RUL) Curve Fitting

Predicts critical decay progression using exponential degradation curves fit on sensor drift velocities:

Health(t) = H_0 * e^-αt
Where:
- H_0 is the starting baseline health index (100%).
- α is the calculated degradation coefficient (calculated dynamically from rolling window regression on sensor deviation changes).
- t is the simulated time.
